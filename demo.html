GitHub Integration Setup

Step-by-Step Setup
  1. Create GitHub Repository: Go to GitHub and create a new repo
  2. Generate Access Token: Settings → Developer settings → Personal access tokens
  3. Configure in App: Go to Exports page and enter credentials
  4. Test Sync: Click "Sync Data to GitHub" to verify connection
Tip: The system automatically backs up case data whenever you add or modify cases!
